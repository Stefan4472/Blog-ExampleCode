This demo shows a simple TextureAtlas implementation. The full article is available at https://stefankussmaul.home.blog/2018/10/14/coding-a-texture-atlas-c-c-sdl/

This code requires a compiler for C++11. It also uses the SDL library (https://www.libsdl.org/). For a good tutorial on how to set up SDL, check out http://lazyfoo.net/tutorials/SDL/01_hello_SDL/index.php
